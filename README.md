# plane-game
🎮 基础操作
方向键 (↑ ↓ ← →)： 控制战机在弹幕中穿梭。

Z 键： 主武器开火 / 确认选择 / 推进剧情对话。

X 键： 释放保险（Bomb）。清除全屏敌方弹幕并对所有敌人造成巨大伤害，释放期间机体完全无敌。

Shift 键： 低速模式。按住时战机移动速度大幅降低，助你进行极限微操，在密集的弹幕缝隙中精准穿梭。

P 键： 暂停/继续游戏。

⚙️ 核心系统与机制
核心判定点： 战机的外形虽然不小，但真正的受击判定点只有机体正中心闪烁的微小核心。只要核心不碰到子弹，机翼擦过火力网也完全安然无恙！

大范围拾取： 战机配备了强大的引力场，无需冒着生命危险贴脸，只要稍微靠近道具即可自动吸附入舱。

🟩 B (Bomb)： 增加一次保险使用次数。

💖 H (Heal)： 恢复 1 点生命值（如果满血时拾取，则转化为 1000 分奖励）。

🟦 蓝点： 击破普通敌机掉落，拾取可获得 100 分。

分数奖励 (Score Extend)： 努力刷分！当你的总得分首次突破 8,000 分和 20,000 分时，总部会直接从天际空投 1 个 Bomb 和 1 点 HP 的强力补给。

时间轴与 Boss 降临： 关卡 Boss 会在倒计时结束后准时降临。但请注意，你击破的小怪越多，推进速度越快，Boss 就会越早现身！
